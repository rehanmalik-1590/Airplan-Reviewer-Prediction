ğŸ“Š Dataset
The dataset contains 103,904 customer reviews with 27 features including:

Demographic information: Gender, Age, Age Band

Travel details: Type of Travel, Class, Flight Distance, Destination, Continent

Service ratings: 14 different service aspects (Seat comfort, Inflight entertainment, Food and drink, etc.)

Operational metrics: Departure and Arrival Delay in Minutes

Target variable: Satisfaction (Yes/No)

Logistic Regression

Random Forest

Gradient Boosting

Support Vector Machines

Neural Networks

Performance is evaluated using metrics such as:

Accuracy

Precision, Recall, F1-Score

ROC-AUC Score

Confusion Matrix
